# 医学論文例題作成ガイド

## 概要
例題1-3の確立された形式に従って医学論文読解問題を作成し、構造と評価基準の一貫性を維持します。

## 既存例題の確認
新規例題作成前に、english_questionディレクトリ内の既存例題を必ず確認し、以下の点に注意してください：

1. 重複の防止
   - 既存例題で扱われている疾患領域の確認
   - 使用されている研究手法や治療アプローチの確認
   - 診断技術や治療戦略の重複を避ける
   - 分子メカニズムの重複を避ける

2. 新規例題作成の方向性
   - 未カバーの医学分野の選択
   - 異なる研究アプローチの採用
   - 新しい技術や方法論の導入
   - 独自の臨床的意義の提示

## ファイル構造要件

### 1. 問題ファイル（例題_{番号}_問題.md）
```
# Example Question {N}: [タイトル] ([単語数] words)
# [日本語タイトル]（[単語数]語）

## Original Text / 英文
[段落番号付きの英文]

## Questions / 設問
1. [第1問]
2. [第2問]
3. [第3問]
```

### 2. 解答・解説ファイル（例題_{番号}_解答解説.md）

## 一貫性のガイドライン

1. 文章選択
   - 例題間で同程度の長さ（600-800語）
   - 同等の難易度レベル
   - 最新の医学研究に焦点
   - 明確な方法論と結果

2. 設問形式
   - 例題間で類似の設問タイプ
   - 以下の要素を適切に組み合わせて設問を構成：
     * 研究背景・目的の理解
     * 研究手法の詳細と妥当性
     * 結果の解釈と意義
     * 研究の革新性
     * 臨床応用の可能性
     * 今後の展望
   - バランスの取れた難易度の進行

3. 解答形式
   - 模範解答の詳細度を統一
   - 研究手法に関する説明を含める
   - 設問ごとの要点数を統一
   - 統一された評価基準

4. 言語サポート
   - 同程度の語彙項目数
   - 一貫した文法ポイントの形式
   - 並行した読解ストラテジーのガイダンス
